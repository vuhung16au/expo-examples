# How to Run

1. Open a terminal and navigate to the project directory:
   ```sh
   cd with-custom-font
   ```
2. Install dependencies:
   ```sh
   npm install
   ```
3. Ensure your `package.json` contains the following scripts section:
   ```json
   "scripts": {
     "start": "expo start"
   }
   ```
4. Start the Expo development server:
   ```sh
   npx expo start
   ```
